# Rsbuild React + Typescript Boilerplate

## Features

`react`

`typescript`

`eslint`

`prettier`

`absolute paths with '@/*'`

## Setup

Install the dependencies:

```bash
npm install
```

## Get Started

Start the dev server:

```bash
npm run dev
```

Check eslint issues:

```bash
npm run eslint
```

Fix all eslint issues:

```bash
npm run eslint:fix
```

Build the app for production:

```bash
npm run build
```

Preview the production build locally:

```bash
npm run preview
```
